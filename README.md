## Description
Automated job monitoring and alert tool built with Python.

The project collects job listings, processes them based on predefined criteria, and helps streamline the job search workflow.  
Designed as a practical automation tool rather than a full production system.

## Features
- Job data collection (scraping)
- Basic filtering and processing
- Automated letter generation
- Script-based execution
- Optional containerized setup (Docker)

## Project Structure
- `scraper_indeed.py` – job scraping logic
- `main.py` – main execution flow
- `generate_letter.py` – simple letter generation
- `api.py` – helper API logic
- `run_bot.bat` – local execution script

## Tech Stack
Python, Requests, BeautifulSoup, Docker

## Notes
This project is provided for demonstration and portfolio purposes.
